## ğŸ¯ What's New

### Automatic Target Detection

SmartEDA now automatically detects your target column - no need to specify it manually!

**Before (v1.0.5)**:
```python
from essentiax.eda import SmartEDA

eda = SmartEDA()
report = eda.analyze(df, target='your_target_column')  # Had to specify
```

**Now (v1.0.6)**:
```python
from essentiax.eda import SmartEDA

eda = SmartEDA()
report = eda.analyze(df)  # Auto-detects target! âœ¨
print(f"Detected target: {eda.detected_target}")
```

### How It Works

EssentiaX uses the built-in `SmartVariableDetector` to analyze your dataset and automatically identify:
- The most likely target column based on data characteristics
- Column types (numeric, categorical, datetime, text)
- Meaningful features vs. ID columns
- Problem type (classification, regression, NLP)

### Benefits

1. **Zero Configuration**: Just load your data and go!
2. **Smart Detection**: Uses AI-powered heuristics to find the target
3. **Still Flexible**: Can override with manual target if needed
4. **Seamless Integration**: Works with all downstream features (AutoML, Feature Engineering)

### Example 3: Use Detected Target in Pipeline
```python

from essentiax.feature_engineering import FeatureEngineer
from essentiax.automl import AutoML

# Step 1: EDA with auto-detection
eda = SmartEDA()
report = eda.analyze(df)

# Step 2: Use detected target for feature engineering
target_col = eda.detected_target
X = df.drop(target_col, axis=1)
y = df[target_col]

# Step 3: AutoML
X_new = FeatureEngineer().fit_transform(X, y)
automl = AutoML(task='classification')
automl.fit(X_new, y)

print(f"âœ… Complete pipeline with auto-detected target: {target_col}")
```

## ğŸ’¡ Technical Details

### Auto-Detection Algorithm

The `SmartVariableDetector` analyzes:

1. **Column Cardinality**: Low cardinality suggests categorical target
2. **Data Types**: Numeric vs. categorical vs. text
3. **Value Distribution**: Balanced vs. imbalanced classes
4. **Column Position**: Last column often the target
5. **Naming Patterns**: Columns named 'target', 'label', 'y', etc.
6. **Correlation**: High correlation with other features
7. **Missing Values**: Targets usually have fewer missing values

### Confidence Score

The detector assigns a confidence score to each potential target column and selects the highest-scoring one.
