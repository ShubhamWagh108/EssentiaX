## ✨ New Visualization Types

### 1. 3D Bubble Scatter
**YouTube-style structure analytics**

```python
from essentiax.visuals.pro_analytics_viz import create_3d_bubble_scatter

create_3d_bubble_scatter(
    df,
    x_col='sentiment',
    y_col='engagement_rate',
    z_col='title_length',
    size_col='views',
    color_col='views',
    dark_theme=True
)
```

**Features**:
- Interactive 3D rotation
- Bubble sizes represent magnitude
- Color gradients for additional dimension
- Professional dark theme

---

### 2. Sunburst Hierarchy Charts
**Interactive nested circular charts**

```python
from essentiax.visuals.pro_analytics_viz import create_sunburst_hierarchy

create_sunburst_hierarchy(
    df,
    path_columns=['sentiment', 'category', 'subcategory'],
    value_column='views',
    color_column='engagement_rate',
    dark_theme=True
)
```

**Features**:
- Multi-level hierarchy
- Click to zoom into segments
- Color-coded by any metric
- Beautiful radial layout

---

### 3. Professional Correlation Heatmap
**Annotated correlation matrix**

```python
from essentiax.visuals.pro_analytics_viz import create_correlation_heatmap_pro

create_correlation_heatmap_pro(
    df,
    columns=['views', 'likes', 'comments', 'sentiment'],
    annotate=True,
    dark_theme=True
)
```

**Features**:
- Annotated correlation values
- Red-Blue diverging colorscale

## 💡 Quick Start

```python
import pandas as pd
from essentiax.visuals.pro_analytics_viz import pro_analytics_dashboard

# Load your data
df = pd.read_csv('your_data.csv')

# Create professional dashboard
pro_analytics_dashboard(df, config='auto', dark_theme=True)
```

## 🎨 Dark Theme

All visualizations feature a professional dark theme optimized for presentations:

- Background: `#1a1a1a`
- Grid: `#333333`
- Text: White
- Accent colors: Cyan, Red, Teal, Yellow

Toggle with `dark_theme=True/False`
